# Kaapi Culture Paid Ads Performance Report

## Overview

This project is a responsive HTML dashboard that presents the paid advertising performance of **Kaapi Culture** for **July 2026**. The report analyzes campaign performance across **Meta Ads** and **Google Ads**, providing key performance indicators, campaign comparisons, and strategic recommendations.

## Features

- Responsive HTML and CSS design
- Executive summary dashboard
- KPI cards
- Meta Ads performance analysis
- Google Ads performance analysis
- Campaign comparison tables
- Strategic insights
- Performance charts
- Professional report layout

## Technologies Used

- HTML5
- CSS3
- Responsive Design
